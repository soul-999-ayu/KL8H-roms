Disclaimer & Prerequisites

Warning: This is an advanced method. Flashing a custom ROM may brick your device if the guide is not followed properly. This process will wipe all your data. Proceed at your own risk.

Before you begin, ensure you have:

  • An unlocked bootloader on your Tecno device (the "target" phone).
  • A **second Android phone** (the "host" phone).
  • The ROM .zip file extracted into the Downloads folder of your host phone.
  • Termux app installed on your host phone.
  • An OTG cable or USB-C to USB-C cable to connect the phones.

Part 1: Setting Up Your Termux Environment

1. Open Termux on your host phone.

2. First, install ADB & Fastboot within Termux by running this command:

curl -s https://raw.githubusercontent.com/offici5l/termux-adb-fastboot/main/install | bash
IMPORTANT - Move The Files: For security reasons, Android prevents running scripts directly from the Downloads folder. You must copy the ROM files into Termux's private directory first.

3. Let's assume you extracted the ROM zip to a folder named KL8h_ROM inside your Downloads. Run these commands to copy it:

# Setup termux storage
termux-setup-storage
                    
# Go to your phone's main storage
cd ~/storage/downloads

# Copy the entire ROM folder to a new 'flasher' directory in Termux
cp -r KL8h_ROM ~/flasher

# Navigate into the new, working directory
cd ~/flasher

Part 2: Flashing the ROM

1. Enable USB Debugging on your Tecno phone and connect it to your host phone.

2. In the Termux terminal (make sure you are in the ~/flasher directory), run the installer script:

First, you need to make the script executable. This only needs to be done once. 
chmod +x *.sh

3. Now, run the installer:

./android_installer.sh

Follow the on-screen instructions provided by the script. Once it finishes, your phone will reboot. The first boot can take several minutes.

Part 3: Post-Flash Setup & Patching

IMPORTANT: After your device boots, complete the initial setup. Then, you must re-enable Developer Options. Go to Settings > About phone and tap the Build number 7 times.
CRITICAL STEP: Before patching, go to Settings > System > Developer options and enable both USB debugging and Rooted debugging. The script will fail otherwise.

1. With your phone booted and debugging enabled, connect it to your host phone.

2. In Termux, from the ~/flasher directory, run the patching script:

./android_patch.sh

Follow the instructions carefully. After the script is done, your device is fully set up and will reboot one last time!
