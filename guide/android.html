Disclaimer & Prerequisites

Warning: Flashing a custom ROM can brick your device. This process will completely wipe all your data. Proceed at your own risk. Always have a backup!

Before you begin, ensure you have:

  • An unlocked bootloader.
  • The ROM .zip file extracted on your second phone.
  • Bugjaeger installed on your second phone.
  • USB Debugging enabled on your Tecno device (before flashing).
  • A USB-C to USB-C cable or a USB-A cable with an OTG adapter.

Part 1: Wiping the Device

First, connect your Tecno phone to the phone running Bugjaeger and authorize the connection. Then, run the following commands in the Bugjaeger shell.

Reboot to fastboot mode:

adb reboot fastboot

Once in fastboot, wipe data and all necessary partitions (if any of these after erasing system fails, you can continue to next step without any issue):

fastboot -w
fastboot erase system
fastboot delete-logical-partition product_cow
fastboot delete-logical-partition product_a
fastboot delete-logical-partition product
fastboot delete-logical-partition system_ext
fastboot delete-logical-partition system_ext_a

Part 2: Flashing the New System

Flash the main system image. Bugjaeger will prompt you to select the file. Choose system.img from the resources/images folder of your extracted ROM.

fastboot flash system

Part 3: Flashing Boot & Root (Choose ONE)

Follow the set of commands for your desired root method. Do not mix them.

Option A: For Magisk Root

fastboot flash boot
# Select boot_5.15.192.img from resources/images folder

fastboot reboot bootloader

fastboot flash init_boot_a
# Select init_boot_a_magisk.img from resources/device/YOUR_DEVICE folder

Option B: For KernelSU (KSU) Root

fastboot flash boot
# Select boot_5.15.192.img resources/images folder

fastboot reboot bootloader

fastboot flash init_boot_a
# Select init_boot_a_ksu.img from resources/device/YOUR_DEVICE folder

Option C: For APatch Root

fastboot flash boot
# Select boot_5.15.192_apatch.img resources/images folder

fastboot reboot bootloader

fastboot flash init_boot_a
# Select init_boot_a.img from resources/device/YOUR_DEVICE folder

Option D: No Root / Kernel Upgrade Only

fastboot flash boot
# Select boot_5.15.192.img resources/images folder

fastboot reboot bootloader

fastboot flash init_boot_a
# Select init_boot_a.img from resources/device/YOUR_DEVICE folder

Part 4: Disabling AVB Verification

This is a crucial step to prevent bootloops. Select the files from the resource/device/spark folder when prompted.

fastboot flash vbmeta_a --disable-verity --disable-verification
# Select vbmeta_a.img from resources/device/YOUR_DEVICE folder

fastboot flash vbmeta_system_a --disable-verity --disable-verification
# Select vbmeta_system_a.img from resources/device/YOUR_DEVICE folder

Part 5: Finalizing and Rebooting

Run a final wipe and reboot your device. The first boot can take 5-15 minutes, so be patient!

fastboot -w
fastboot reboot

Part 6: Post-Flash Setup & Fixes

IMPORTANT: After your device boots into the new ROM, you must re-enable Developer Options. Go to Settings > About phone and tap the Build number 7 times.
CRITICAL STEP: Before running any commands below, go to Settings > System > Developer options and enable both USB debugging and Rooted debugging. The post-install scripts will fail otherwise.

These commands will apply necessary fixes for features like VoLTE and set correct permissions.

1. First, push the required fix files from your Bugjaeger phone to your Tecno device. You will need to select the files from your extracted ROM folder (e.g., resources/fixes/).

adb push resources/fixes/fixes.tar.gz /sdcard/
adb push resources/fixes/service.sh /sdcard/

2. Now, run the following shell commands to apply the fixes:

adb root
adb shell am start -n me.phh.treble.app/.SettingsActivity

Wait for about 3 seconds for the app to initialize, then continue.

adb shell am force-stop me.phh.treble.app
adb shell tar -xzvf /sdcard/fixes.tar.gz -C /
adb shell chown -R system:system /data/data/me.phh.treble.app
adb shell restorecon -Rv /data/data/me.phh.treble.app
adb shell rm /sdcard/fixes.tar.gz
adb shell setprop persist.sys.phh.restart_ril true
adb shell "mkdir -p /data/adb/service.d && mv /sdcard/service.sh /data/adb/service.d/service.sh && chmod +x /data/adb/service.d/service.sh"

3. Install the essential apps. Bugjaeger will prompt you to select the APKs from your extracted ROM folder (e.g., resources/apps/).

adb install resources/apps/ims.apk
adb install resources/apps/gcam.apk
adb install resources/apps/photos.apk

4. Finally, reboot your device to make sure all changes take effect.

adb reboot

Optional App Installs

These apps are not required but can improve your experience.

Lawnchair Launcher

A clean, customizable, open-source launcher.

adb install resources/apps/lawnchair.apk

YouTube & Music (Premium)

Install MicroG first, then the other apps.

adb install resources/apps/microg.apk
adb install resources/apps/youtube.apk
adb install resources/apps/music.apk
